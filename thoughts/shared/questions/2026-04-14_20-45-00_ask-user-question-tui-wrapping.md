# Research Questions: ask-user-question TUI text wrapping

## Discovery Summary

The `ask-user-question` tool is published as the external npm package `@juicesharp/rpiv-ask-user-question@0.1.2`, with local source at `/Users/sguslystyi/rpiv-ask-user-question/` and installed copy at `/usr/local/lib/node_modules/@juicesharp/rpiv-ask-user-question/`. It is referenced from rpiv-pi only in `extensions/rpiv-core/package-checks.ts` (installation check) and `extensions/rpiv-core/index.ts` (missing-plugin list in `/rpiv-setup`). Discovery confirmed two layers of the truncation bug: (1) `ask-user-question.ts` concatenates `label — description` into one string and constructs `SelectItem` without a `description` field, and (2) `SelectList.renderItem()` in `@mariozechner/pi-tui` truncates every code path with `truncateToWidth` — it never wraps, even though the sibling `Text` component uses `wrapTextWithAnsi` from the same utils module. Prior research at `thoughts/shared/research/2026-04-15_01-19-00_ask-user-question-tui-text-wrapping.md` reached the same diagnosis. Key files read for depth: `ask-user-question.ts` (full), `select-list.js` (full), `select-list.d.ts` (full), `text.js` (full), plus grepped surveys of `utils.js` and rpiv-core references.
